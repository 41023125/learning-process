提取資料

from bs4 import BeautifulSoup
import requests
url = "https://41023125.github.io/learning-process/content/index.html"
response = requests.get(url)
soup = BeautifulSoup(response.coent.decode('UTF-8'), 'html.parser')
span_tags = soup.find_all(['a'])
print(span_tags)

延續前篇加入span_tags = soup.find_all(['a'])

find_all 是 BeautifulSoup 中的一個方法，它用於尋找符合指定條件的所有元素。在這個例子中，條件是 ['a']，表示尋找所有的 <a> 標籤元素。

除了 find_all 方法之外，BeautifulSoup 還提供了其他常用的方法來解析和操縱網頁的內容。以下是一些常見的 BeautifulSoup 方法：

  1. find(tag, attrs, recursive, text, **kwargs)：尋找符合條件的第一個元素，與 find_all 類似，但只返回第一個匹配的結果。
  2. find_next(tag, attrs, recursive, text, **kwargs)：尋找下一個符合條件的元素。
  3. find_all_next(tag, attrs, recursive, text, **kwargs)：尋找之後所有符合條件的元素。
  4. find_previous(tag, attrs, recursive, text, **kwargs)：尋找前一個符合條件的元素。
  5. find_all_previous(tag, attrs, recursive, text, **kwargs)：尋找之前所有符合條件的元素。
  6. select(selector)：使用 CSS 選擇器選擇元素。
  7. get_text()：獲取元素的內容文本。
  8. get(tag, default)：獲取元素的屬性值。
  9. has_attr(name)：檢查元素是否具有指定的屬性。
  10. prettify()：以縮進和換行的方式返回美化後的 HTML 字符串。

<a>標籤是 HTML 中的超連結元素，通常用於創建鏈接到其他網頁或文件的文字或圖片。這行程式碼的目的是從 soup 物件中找到所有的<a>標籤元素。

例如想要抓取全部<a>標籤下的href屬性裡面的內容 

span_tags = soup.find_all('a')
for span_tag in span_tags:
    class_attr = span_tag.get("href")
    if class_attr:
        print(class_attr)

想要抓取<a> 標籤所顯示的文字內容

a_tags = soup.find_all('a')
for a_tag in a_tags:
    text_content = a_tag.get_text()
    print(text_content)

使用 get_text() 方法獲取 <a> 標籤內的文字內容

使用 get_text() 方法時，它會返回所有子節點的文字內容，包括嵌套的子元素。如果只想獲取直接內容，而不包括子元素的文字，可以使用 string 屬性，像這樣：text_content = a_tag.string。
